# Swiggy SQL Analytics Project

A compact SQL case study analyzing Swiggy food-delivery data to understand **what users order, where demand comes from, pricing patterns, and which dishes & restaurants drive engagement**.

---

## What This Project Includes
- Data cleaning & preprocessing  
- Star schema (ERD) design  
- Dimension & fact table creation  
- SQL scripts for modeling + insights  
- KPIs & analysis across cities, restaurants, categories, pricing, and ratings  

---

## Project Structure
```
brd/           → Business Requirements Document  
datasets/      → Raw dataset  
sql-scripts/   → Cleaning, modeling & insights SQL  
erd/           → Entity Relationship Diagram  
```

---

## Key Insights
- Top cities, restaurants, and food categories  
- Price bucket performance (₹0–100 → 500+)  
- Rating distribution & quality trends  
- State-wise and monthly demand patterns  

---

## Tools Used
- SQL (MySQL Workbench)  
- Dimensional Modeling
